# UNIVERSE – University Services Platform

**UNIVERSE** is a comprehensive platform designed to simplify university life by connecting students with essential campus services, tutors, navigation tools, and a student community hub.

## Core Modules
* **Tutoring:** Peer-to-peer academic support.
* **Navigation:** Campus building & facilities mapping.
* **Community:** Social hub for student interaction.
* **Rewards:** Gamified points & volunteer hours tracking.

## Technical Approach
Built using **OOAD methodology** with **UML modeling** for robust architecture. Tested through comprehensive unit and integration strategies to ensure reliability.

## Tools
UML, Visual Paradigm, Lucidchart, Draw.io.
